# Топ-20 миксов для Арома Ателье

Подборка собрана из публичных русскоязычных каталогов миксов
(hookahportal.ru, ohookah.ru, premium-tabak.com, hookahhouse.ru,
nn-kalyan.ru, store.allkalyans.com, 4kalyans.ru). Критерии отбора:
популярность/наличие пометки «Хит», доступность табаков
(Darkside, MustHave, Adalya, Sebero, Spectrum, BlackBurn, Tangiers,
Serbetli, AlFakher, Starbuzz) и разнообразие вкусовых профилей —
покрыто 8 категорий каталога Nomad: фрукты, тропик, ягоды, цитрусы,
десертка, напитки, специи/травы, прочие/с кислинкой.

## Замечания по загрузке в Nomad

1. Таксономия каталога Nomad строго разделяет три поля
   (см. `CLAUDE.md` §7): `flavorProfiles` — категории-«полки»,
   `flavors` — конкретные ноты, `flavorTags` — только мета-теги
   (хит, новичкам, холодок и т. п.). При импорте не смешивать колонки.
2. Бренды и SKU перед загрузкой свериться с инвентарём (`Master/Инвентарь`):
   рекомендации в `Арома Ателье` зависят от наличия табаков
   (`CLAUDE.md` §7, инвариант №3).
3. Гостевой контур не хранит favorites/sessions — миксы загружаются
   как каталог, не как «мои миксы пользователя» (§7, инвариант №1).
4. Пропорции — ориентир. На стороне `Мастера` проверить, что сумма
   процентов = 100 для каждого микса (валидация на границе).

# Предустановленные рейлы Арома Ателье

Seed-набор `prepared`-рейлов для `Мастер → Менеджер рейлов`
(см. `PRD.md` §7 и §9.6). Названия атмосферные, как просил
заказчик («Лёгкие будни», «Забытые империи»), чтобы попадать в
премиальный lounge-character Nomad (`PRD.md` §12 п.4, primary
`#551817`). Источник миксов — `top-20-mixes.md`.

## Типы рейлов в системе

Согласно `PRD.md` §7, любой рейл имеет поле `type`:

| `type` | Назначение | Откуда берётся |
|---|---|---|
| `prepared` | Авторские подборки от Nomad | Seed + `Мастер` |
| `curated` | Сборки кальянных мастеров | `Мастер` |
| `statistical` | Авто-подборки по аналитике | Backend, обновляется по событиям |

Этот файл — только `prepared`. Дополнительно MVP обязан показывать
два `statistical`-рейла (§7.1): «Топ по кнопке Покурить» и «Топ по
оценкам» — они формируются автоматически из `smoke_cta_clicked` и
`MixRating`, наполнение здесь не задаём.

## Детальное содержимое

### `light-weekdays` — Лёгкие будни

> Для перерыва между встречами. Лёгкие 2–3-табачные миксы, которые
> не «забивают» вкус кофе и не тянут разговор в тяжёлый вечер.

Критерий: ≤3 компонентов, ведущая нота — фрукт или цитрус без
сладкой густоты, низкая/средняя крепость, без алкогольных и
табачно-сигарных профилей.

- №3 Тропики
- №10 Melon Cola
- №11 Pineapple Mint
- №12 Bodrum Tangerine
- №13 Молоко-Банан

### `forgotten-empires` — Забытые империи

> Густые, дорогие, узнаваемые. Чай, пряности, ром, тёмная карамель —
> вкусы, которые держат стакан час и не теряют очертаний.

Критерий: премиальный/насыщенный профиль, есть «гастро»-нота
(пряное / чай / десерт / алкоголь), пропорции с акцентом на одну
доминирующую глубину.

- №2 Космокола — кола, цветы, холодок: визитка Darkside
- №4 Итальянское мороженое — базилик и мороженое, плотная гастрономия
- №8 Виноград-Лимон-Имбирь — пряный имбирь держит длинный финиш
- №9 Cinnamon Red Tea — корица + чёрный чай, согревающая классика
- №16 Citrus Brazillian Rum — карибский ром, чай, цитрус: коктейльная глубина
- №17 Cookies Bramble Cinnamon — печенье, ежевика, корица: тяжёлый десерт

### `bodrum-sunset` — Закат в Бодруме

> Курортный вечер у моря. Цитрусовая кислинка, прохлада мяты,
> сочный тропик. Хорошо после еды и перед прогулкой.

Критерий: цитрус или тропик как ведущая нота + мята/арктик/холодок.

- №6 Lemon Pineapple
- №7 Pirate Citrus
- №11 Pineapple Mint
- №12 Bodrum Tangerine
- №18 Bubble Fruit

### `garden-under-snow` — Сад под снегом

> Ягоды и хрустящий мороз. Для тех, кто любит, чтобы во рту
> кололо льдом.

Критерий: ведущая нота — ягода, есть выраженный холодок
(Supernova / Ice Granny / арктик / Dark Icecream).

- №1 Морозная ягода
- №5 Клубничный лёд
- №14 Молоко-Черника
- №18 Bubble Fruit
- №20 Cheesecake Wild Forest

### `grandmother-buffet` — Бабушкин буфет

> Молоко, варенье, пирог из печи. Тёплое десертное, без льда и
> алкоголя. Тот самый ностальгический угол.

Критерий: молочная/сливочная или выпечная база, без выраженного
холодка, среднее тело, дружелюбно к новичкам.

- №13 Молоко-Банан
- №14 Молоко-Черника
- №15 Лимонный пирог
- №17 Cookies Bramble Cinnamon
- №19 Fresh Time

## Замечания по интеграции

1. **Сущности.** `Rail` + `RailItem` (см. `PRD.md` §5.2). Для seed
   достаточно: `type='prepared'`, `slug`, `title`, `subtitle`,
   массив `RailItem` с порядком.
2. **Доступность по инвентарю.** Если хотя бы один табак микса
   `InventoryItem.available=false`, микс выпадает из выдачи рейла
   (`PRD.md` §4 п.8, §6.2 п.1). Рейл не показывать, если в нём
   осталось < 2 доступных миксов.
3. **Главная.** Рейлы видны гостю на `Главной` Aroma Atelier после
   рекомендаций (`PRD.md` §8.5, §7.2 п.1). Порядок:
   `statistical` («Топ по Покурить» → «Топ по оценкам») →
   prepared в порядке таблицы выше.
4. **Без персонализации.** Гостевой контур не хранит избранное и
   историю (`PRD.md` §4 п.3-4). prepared-рейлы одинаковы для всех
   гостей в рамках одного состояния инвентаря.
5. **Редактирование.** Любое последующее изменение состава рейлов —
   через `Мастер → Менеджер рейлов` (`PRD.md` §9.6), не патчем в
   коде Aroma Atelier.

# Nomad Telegram Bot

Сервис для staff-рассылки и выдачи daily code.

## Назначение

Этот сервис должен стать отдельным Nomad worker и не должен встраивать продуктовую бизнес-логику в себя.

На текущем этапе сервис создан как scaffold.

## Локальный запуск

```bash
cd services/nomad-telegram-bot
npm install
npm run dev
```

## Стадия

Текущая стадия: placeholder до подключения Telegram API и Nomad backend contracts.
